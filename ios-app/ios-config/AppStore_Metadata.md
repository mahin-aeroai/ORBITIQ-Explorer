## Description
ORBITIQ Explorer is a real-time space situational awareness platform built for space enthusiasts, students, and aerospace professionals.

Track satellites live. Monitor asteroids passing Earth. Explore active comets. Plan meteor shower viewing. All from a single NASA Mission Control-inspired interface.

**SATELLITE TRACKING**
• International Space Station (ISS) — live latitude, longitude, altitude
• Starlink constellation — 6,000+ satellites
• GPS, NavIC (India's navigation system), Galileo, GLONASS
• Weather satellites: INSAT-3DR, METEOSAT, GOES
• Earth observation: Sentinel-2, Landsat 9, Cartosat-3

**ASTEROID DASHBOARD**
• Near Earth Object (NEO) 7-day feed from NASA
• Potentially Hazardous Asteroid (PHA) alerts
• Close approach distance, velocity, and size data
• Risk classification: LOW / MEDIUM / HIGH

**COMET EXPLORER**
• 6 featured active/notable comets
• Perihelion distance, magnitude, discovery history
• Includes: Tsuchinshan-ATLAS, Halley, 67P, ATLAS C/2024 G3

**METEOR CENTER**
• Full 2026 annual shower calendar (9 major showers)
• Zenithal Hourly Rate (ZHR), velocity, best viewing times
• Recent fireball event log with magnitude and location

**INTERACTIVE 3D GLOBE**
• Animated Earth with ISS live orbit track
• Starlink shell ring and NavIC GSO satellites
• Toggle orbital layers: LEO, MEO, GEO
• ISS visible pass predictions

**SEARCH**
• Unified search across satellites, asteroids, comets, and meteor showers

Data sources: CelesTrak · NASA NEO API · JPL Small Body Database · IMO
